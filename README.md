# Test Store

A test store application for Insights Pixel Integration (Manual).

Simple e-commerce site with 3 tea pot products. Features product listing, detail pages, and cart/wishlist functionality using localStorage.

**Structure:**
- `/` - Landing page
- `/products/` - Product listing
- `/products/1|2|3/` - Product detail pages
